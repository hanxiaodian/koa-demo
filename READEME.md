## koa-demo

### 技术选型：koa2 + mysql + sequelize + jest + swagger

### 底层依赖：
    - node         v14.17.5
    - npm          v6.14.14
    - mysql        v8.0.26
    - nodemon   v2.0.12(根据文件改动重新运行项目)
    - ...


### 运行
    - npm install
    - 更改本地 mysql 配置（src/lib/database）（后续会针对不同环境设置配置文件）
    - npm run dev
    - npm run test (测试)
    - npm run lint  (检测代码规范并自动修改可修改的 code)
